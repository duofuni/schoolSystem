                    <li>缓存：HTML5 web</li>

                    <li>css框架：less,rem</li>
                </ul>
            <div>
            <div>

                    <li>总体：express</li>
                    <li>数据库：MongoDB</li>
                    <li>MongoDB对象模型工具：mongoose</li>
                    <li>图片上传：Multer中间件</li>
                    <li>加密：bcrypt</li>
                    <li>跨域认证解决方案：jsonwebtoken</li>
                </ul>
